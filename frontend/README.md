# Prompt-to-SQL Premium UI

A production-grade, AI-powered data analytics interface built with React, Vite, Tailwind CSS, and Recharts.

## 🚀 Key Features

*   **⚡ Modern Chat Interface**: Natural language querying with streaming state feedback.
*   **📊 Interactive Response Cards**: Multi-view cards (Table, dynamic Charts, SQL).
*   **🌓 Dual Theme Architecture**: Seamless transitions between Dark (default) and Light modes.
*   **🛠️ SQL Transparency**: View, copy, and analyze generated SQL in a terminal-themed UI.
*   **📜 History Tracking**: Quick access to past queries and results.
*   **✨ Premium Micro-interactions**: Smooth transitions, glassmorphism, and responsive design.

## 🛠️ Tech Stack

*   **React + TypeScript + Vite**: Frontend core.
*   **Tailwind CSS**: Custom design system and components.
*   **Recharts**: Advanced data visualization.
*   **Zustand**: Lightweight state management.
*   **Framer Motion**: Interface animations.
*   **Lucide React**: Vector icons.

## 📦 Getting Started

1.  `cd frontend`
2.  `npm install`
3.  `npm run dev`

## 🎨 Theme Configuration

Colors and tokens are defined in `src/index.css` using CSS variables, allowing for real-time theme switching.

*   **Dark Mode**: Professional, high-contrast dark palette (#0B0F17).
*   **Light Mode**: Clean, airy, and high-readability palette (#F9FAFB).
